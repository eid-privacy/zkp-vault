# Frameworks

Overview of ZKP proof system families and how to compare them. See [index.md](./index.md) for the full breakdown.

## Categories

- **Polynomial IOP-based**: PLONK, Groth16, Sonic, Marlin
- **Transparent / hash-based**: FRI, zk-STARKs, Bulletproofs
- **Trusted setup**: Groth16, Pinocchio

## Comparison Axes

Proof size · Prover time · Verifier time · Setup type (trusted / universal / transparent) · Security assumptions · Language support

> Detailed per-system notes live in [ProofSystems/](../ProofSystems/README.md). This directory provides the comparison framework and evaluation criteria (see `ZKP-frameworks.md` in the repo root).
