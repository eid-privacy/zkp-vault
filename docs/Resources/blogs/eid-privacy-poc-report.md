# Proof-of-Concept for ZKPs (eid-privacy, 2026)

## Summary
Documents the Innosuisse Milestone 2 proof-of-concept comparing ZKP systems for privacy-preserving e-ID credentials. Benchmarks cover device/holder binding, issuer signatures, attribute predicates (equality, comparison, revelation), non-revocation, unlinkability, and data minimisation. Systems evaluated: BBS signatures, Bulletproofs (via Dock Network), VB accumulators, and UltraHONK (Noir/Aztec). Identifies Longfellow as the most promising overall approach.
