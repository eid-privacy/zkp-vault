# Resources on Zero-knowledge Systems and Proofs (eid-privacy, 2025)

## Summary
Curated reading list for zero-knowledge proofs organised by complexity level — from introductory cryptography courses (Christopher Paar, MIT) up to advanced research material (Justin Thaler's Proofs, Arguments, and Zero-Knowledge; Alessandro Chiesa's summer school). Includes pointers to practical workshops (0xparc / Circom), STARK-101, and the MoonMath Manual.
