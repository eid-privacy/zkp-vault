# Open Source SWIYU Demo application (eid-privacy, 2025)

## Summary
Announces the open-source proof-of-concept web application for Switzerland's Swiyu e-ID infrastructure, deployed at https://eid.c4dt.org. Implements the issuer workflow; verifier workflow and advanced credential verification are in development. Source code at github.com/eid-privacy/SWIYU-Demo.
