# Halo2 Documentation (Zcash)

## Summary
Official documentation for Halo2, Zcash's PLONKish proof system implementing recursive proofs without a trusted setup. Covers the PLONKish arithmetization model (columns, gates, permutations), the IPA polynomial commitment scheme, and the circuit API. Halo2 is the backend for Zcash Orchard and many Ethereum ZK projects.
