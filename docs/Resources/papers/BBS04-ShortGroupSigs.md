# Short Group Signatures (Boneh, Boyen, Shacham 2004)

## Summary
Introduces BBS (Boneh-Boyen-Shacham) group signatures using pairing-based cryptography, yielding signatures of only a few hundred bytes. The scheme allows a group member to sign messages anonymously; a designated opener can later revoke anonymity. The BBS signature scheme underlies BBS+ and the anonymous credential systems implemented in the Dock Network crypto library.
