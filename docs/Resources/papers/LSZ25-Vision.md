# Vision: A Modular Framework for Anonymous Credential Systems (Lehmann, Sidorenko, Zacharakis 2025)

## Summary
Proposes a modular blueprint for building anonymous credential systems from three interchangeable core components: a commitment scheme, a signature scheme, and a NIZK proof system. Demonstrates that most deployed and proposed anonymous credential systems (including Privacy Pass and EU Digital Identity wallet variants) share this common blueprint, enabling systematic comparison and substitution of components.
